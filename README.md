# 自平衡两轮车 - STM32平衡车控制项目

本项目基于STM32F10x系列微控制器，参考B站UP主“江协科技”的平衡车教程，自主编写并优化了控制代码，实现了小车的自平衡、遥控移动、定距前进、定角转向等功能。项目封装了多个模块（如电机、编码器、MPU6050、NRF24L01、PID控制器等），便于移植和扩展。

## 功能特点

- **自平衡控制**  
  使用MPU6050获取角度和角速度，通过PID控制器（角度环）实现小车的直立平衡。

- **运动控制**  
  - 无线遥控（NRF24L01 / 蓝牙）  
  - 按键控制（定距前进 / 定角转向）  
  - 速度环 + 位置环 + 转向环 + 转向角度环嵌套控制

- **模块化设计**  
  各硬件驱动（电机、编码器、MPU6050、NRF24L01、OLED、串口等）和PID算法均封装为独立模块，易于移植到其他项目。

- **调试支持**  
  支持通过蓝牙串口动态调整PID参数（配合江协科技蓝牙串口小程序），并可通过OLED实时查看关键数据。

## 硬件清单

| 模块              | 型号/说明                     |
| ----------------- | ----------------------------- |
| 主控              | STM32F103C8T6（或其他F10x系列）|
| 电机驱动          | TB6612 / L298N 等             |
| 直流减速电机      | 带霍尔编码器（减速比9.27666） |
| IMU传感器         | MPU6050（I2C接口）            |
| 无线通信（可选）  | NRF24L01 / 蓝牙模块（HC-05）   |
| 显示模块（调试用）| 0.96寸OLED（I2C接口）         |
| 电源              | 12V / 7.4V锂电池（视电机而定） |

> 引脚映射请参考各模块的 `.h` 文件中的宏定义。

## 软件架构

| 模块          | 文件                  | 说明                                 |
| ------------- | --------------------- | ------------------------------------ |
| 系统主控      | `main.c`              | 主循环，任务调度，模式切换           |
| 定时器中断    | `Timer.c`             | 1ms系统节拍，10ms角度环，50ms速度环  |
| PID控制器     | `PID.c` / `PID.h`     | 位置式PID，支持微分先行、积分限幅    |
| 电机驱动      | `Motor.c` / `PWM.c`   | 电机方向控制和PWM输出                |
| 编码器读取    | `Encoder.c`           | TIM定时器编码器模式，获取转速         |
| MPU6050       | `MPU6050.c`           | 读取角度、角速度，互补滤波            |
| 无线通信      | `NRF24L01.c`          | 数据收发（基于江协科技源码）          |
| 蓝牙调试      | `Serial.c`            | 串口中断接收，数据包解析              |
| 显示          | `OLED.c`              | 屏幕显示（基于江协科技源码）          |
| 按键          | `Key.c`               | 按键状态机，支持长按/短按             |

## 运行效果

- 小车可自行保持直立，并响应遥控器的前进、后退、转向指令。
- 切换到“定距前进”模式后，小车会按照预设距离（如10圈编码器计数）精确移动并自动停止。
- 切换到“定角转向”模式后，小车会根据目标转向角度自动旋转并停止。
- 可通过蓝牙串口发送 `[slider,AngleKp,4.0]` 等指令实时调节PID参数。

## 如何编译与烧录

1. 使用 Keil MDK 打开工程（确保已安装STM32F10x器件支持包）。
2. 根据实际使用的STM32型号和晶振频率，修改 `system_stm32f10x.c` 和工程配置。
3. 根据硬件连接，调整各模块 `.h` 文件中的引脚定义（如 `MyI2C.h`, `Motor.h`, `NRF24L01_Define.h` 等）。
4. 编译并烧录到STM32。
